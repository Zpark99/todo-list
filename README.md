# 📌 프로젝트 이름

## ✨ 프로젝트의 한줄 설명
간단한 할 일 목록을 만들고 관리할 수 있는 Todo List 웹 애플리케이션입니다.

## 🔧 기능 및 구조
- 할 일 추가 / 수정 / 삭제
- 각 Todo 카드 형태로 렌더링
- 입력 폼과 리스트 영역 분리

## 💾 데이터 저장 방식
- 로컬스토리지 사용  
(※ 백엔드 연동은 없음)

## 🖥 주요 페이지 구성
- 메인 페이지: Todo 입력, 리스트 출력
- (필요 시 추가)

## 🛠 사용 기술 스택
- HTML / CSS / JavaScript
- React (Vite 기반)
